# ESPP Plan Modeler — Section 423

A dynamic, browser-based Employee Stock Purchase Plan (ESPP) modeling tool for US participants. Built as a single self-contained HTML file — no server, no dependencies, no build step.

## Features

- **Plan Configuration** — Fully editable plan parameters (discount rate, offering period, lookback provision, contribution limits, §423 compliance)
- **Purchase Calculator** — Per-period share purchase modeling with paycheck deduction schedule
- **Multi-Period Projection** — 2–10 year wealth accumulation projections with interactive charts
- **Tax Analyzer** — Side-by-side qualifying vs. disqualifying disposition comparison with full tax breakdown
- **Scenario Analysis** — Bull, flat, and bear market simulations
- **§25K Limit Calculator** — IRS Section 423 annual limit modeling with excess refund tracking

## Quick Start

Open `index.html` in any browser. No installation required.

## Deployment

Hosted on GitHub Pages. Visit the live site or clone and customize for your own ESPP plan.

## Pre-loaded Plan

Defaults are configured from the **Truist Financial Corporation** ESPP prospectus (effective January 1, 2025), but every parameter is editable to model any Section 423 or non-qualified ESPP.
